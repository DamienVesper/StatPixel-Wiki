# Guild
---
Displays guild data for games, insights and more. There are several subcommands that sprout from this command.

---
View information about a guild.
```
=guild lookup [name]
```
---
View game stats in a guild.
```
=guild games [name]
```
---
View guild insights.
```
=guild insights [name]
```
---
View a member's stats in a  guild.
```
=guild member [name]
```
